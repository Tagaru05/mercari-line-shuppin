# 📦 メルカリ出品おまかせライン（LINEに写真を送るだけ）

スマホのLINEに商品の写真を送ると、AIが出品の準備
（背景をきれいにした写真・タイトル・説明文・値段）を整えて、
メルカリの**下書き**まで作ってくれる仕組みを、あなたのMacに構築します。

**最後の「出品ボタン」だけは、必ずあなたが押します。** AIが勝手に売ることはありません。

> ⚠️ **Mac専用**（Windowsでは動きません）。フリマの規約・仕様変更で使えなくなる可能性があります（利用は自己責任）。

---

## 始め方（2ステップ）

1. このリポジトリを手元に落とす（緑の **Code → Download ZIP**、または `git clone`）
2. そのフォルダで **Claude Code** を開いて、こう伝える：

   > **「このフォルダの『セットアップ指示書.md』を読んで、その通りにセットアップを進めて。私は初心者なので、私がやる操作は1つずつ案内してください」**

あとは Claude Code が案内してくれます。一番むずかしいLINEの設定も、**画面操作はClaude Codeが代行**します。
あなたがやるのは、機械には代われない**スマホに届く認証コードの入力・QRでの友だち追加・各種ログイン・許可ボタン**だけ（60〜90分）。

> 💡 この代行には **Claude in Chrome 拡張** が必要です（下の「必要なもの」参照）。これが無いと画面操作を代われないので、必ず先に入れてください。

---

## 必要なもの

- Mac ＋ Claude Code（課金プラン）＋ Claude in Chrome 拡張
- ChatGPT アカウント（画像づくり用。無料でも可・たくさん使うなら Plus 推奨）
- スマホの LINE ／ メルカリのアカウント（Chromeでログイン）
- メールアドレス（無料サービスの登録用）

## 中身

| ファイル | 役割 |
|---|---|
| はじめにお読みください.md | 人間が最初に読む案内 |
| **セットアップ指示書.md** | あなたのClaude Codeが読んで実行する作戦書 |
| 運用手順.md | 出品仕込みの詳細手順（Claude Code用） |
| GPTs指示文.md | 商品写真をきれいにするGPTsの中身（予備・自作用） |
| 検証チェックリスト.md | うまく動くか確かめるチェック表 |
| worker/ | 受付サーバーのコード（worker.js）＋ 白抜きスクリプト（shironuki.py）|

## 仕組み（ざっくり）

```
📱 LINEに写真を送る
   → 受付サーバー(Cloudflare・無料)が保存
   → Claude Code が：背景の白抜き ＋ 商品写真づくり(GPTs) ＋ 相場しらべ ＋ 下書き作成
   → LINEに「準備できました」の通知
👤 あなた：メルカリアプリで確認して出品ボタン
```

## 安全のこと

- 秘密の鍵やトークンは `.env` に入れ、GitHubには上げません（`.gitignore`で除外済み）
- 受付サーバーは**あなた本人のLINEからの写真だけ**を受け付けます（他人が友だち追加しても無視）
- 写真は一定期間で自動削除されます（未処理30日・処理済み7日）
