# 📦 LINEに写真を送るだけ・メルカリ出品おまかせセット

> ⚠️ **このセットはMac専用です。Windowsパソコンでは動きません。**

写真をLINEに送ると、AIが出品の準備（きれいな写真・タイトル・説明文・値段）を
ぜんぶ整えて「下書き」まで作ってくれる仕組みを、あなたのパソコンに作ります。

**最後の「出品ボタン」だけは、必ずあなたが押します。**
AIが勝手に売ることは絶対にありません。安心してください。

## 用意するもの（先にチェック）

- [ ] パソコン（Mac）と Claude Code（課金プラン）
- [ ] ChatGPT のアカウント（無料でもOKですが、画像づくりは1日数枚まで。たくさん出品するならPlusがおすすめ）
- [ ] LINEアプリ（いつものスマホのLINE）
- [ ] メルカリのアカウント（パソコンのChromeでログインしておく）
- [ ] メールアドレス1つ（無料サービスの登録2回に同じアドレスを使えます）

## かかる時間

- ぜんぶで **60〜90分**（休憩しながらでOK。途中でやめても壊れません）
- そのうち、あなたが自分でやるのは「アカウント登録・ログイン」だけ。
  残りはぜんぶ Claude Code がやってくれます。

## やり方（これだけ）

1. Claude Code を開く
2. こう伝える：
   **「このフォルダの『01_セットアップ指示書.md』を読んで、その通りにセットアップを進めて。私は初心者なので、私がやる操作は1つずつ案内してください」**
3. あとは Claude Code の案内に従うだけ。ログインなど「あなたにしかできないこと」だけ順番にお願いされます

## 困ったら

- わからなくなったら、Claude Code にそのまま「わからないので解説して」と言えば大丈夫
- 何度聞いてもAIは嫌な顔をしません
- **この手順の中では、押しても壊れません。**失敗したら同じ手順をやり直せばOKです

## 1つだけ正直なお知らせ

メルカリのルールが将来変わって、この仕組みが使えなくなる可能性はあります。
その時は「出品ボタンと同じように、文章コピペも自分の手で」に戻せば大丈夫です。
